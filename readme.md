#Coffee E-Commerce App

##Overview
*This is a coffee ecommerce app where you can order coffee and pay with a bank card.

### Technologies, Frameworks, and Programming Languages Used
* HTML5
* CSS3
* Bootstrap
* JavaScript
* jQuery
* MongoDB w/Mongoose
* AngularJS
* Express
* Node.js
